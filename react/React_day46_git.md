### はじめに
#### 新しいリポジトリの初期化: `git init`
新しいリポジトリを作成するには、git init コマンドを使用します。git init は、新しいリポジトリの初期セットアップ時に使用するワンタイム コマンドです。このコマンドを実行すると、現在の作業ディレクトリ内に新しい .git サブディレクトリが作成されます。これによって、新しい main ブランチも作成されます。
